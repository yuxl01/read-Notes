#### git


1.全局配置默认仓库配置

|git config  --global 'user.name' 'name'|git config global user.email '87717663@qq.com'|
|-|-|
|git config user.name '修改名字' |git config user.email '修改邮箱'|


2.初始化仓库

|操作|命令|
|-|-|
|初始化仓库|git init|
|添加到缓存区|git add 1.txt|
|添加到仓库|git commit -m '提交说明'|
|查看状态|git status|
|查看修改内容|git diff  |
