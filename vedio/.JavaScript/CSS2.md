# css Ⅱ

### 一.常用css属性

###### 1.em
    
    1em = 1*font-size(自身);


```.css
//单行文本垂直居中
line-height高度等于容器高度

//首行缩进
text-indent:2em;

//文字加横线
text-decoration: line-through;

//去掉横线
text-decoration: none;

//下划线
text-decoration: underline;

//鼠标移入展示图标
cursor: copy;

//伪类选择器hover 鼠标移入
a:hover{
  background-color: orange;
}

//圆角
border-radius: 10px;
```
