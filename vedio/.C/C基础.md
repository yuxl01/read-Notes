# 编译执行过程

### 一、C语言源码文件编译

###### .c预处理(gcc -E a.c)
  
    1、展开头文件
    2、删除注释
    3、条件编译
    4、宏定义展开
    
###### 编译
    1、检查语法
    2、将高级语言转为汇编
   
###### 汇编
    1、将汇编语言转为机器语言二进制

###### 链接 (gcc a.o -o a.exe)
    1、链接不同操作系统库文件,可执行文件
    
    
 ### 二、程序执行
     内存四区： 1、代码区 2、数据区 3、栈区 4、堆区
     
    1、从硬盘加载内存中
    2、内存与cpu交互并进行处理
