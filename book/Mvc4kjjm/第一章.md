* # 第一章 ASP.NET + MVC


### 一、传统MVC模式 

    1.将所有与 UI 相关的操作揉合在一起.
    2.包括 UI 界面的呈现、用于交互操作的捕捉与响应、业务流程的执行以及对数据的存取.
  
#### 1.自治视图 (Autonomous View, AV)

        采用了事件驱动的开发方式,所有与UI相关的逻辑部可以定义在针对视图的后台代码中
     并最终注册到视图本身或者视图元素(控件)的相应事件上。
        
- ###### `典型应用`
   - 1.Windows Forms
   - 2.ASP.NET Web Forms
    
- ###### `弊端`
   - 1.业务逻辑和UI是无关的,业务和ui可以被抽象复用,但是在自治视图中ui完全丧失抽象复用功能.
